# Scripts

This directory contains wrappers and runtime helper areas for the Proteus
plugin.

Wrappers:

```text
proteus.ps1
proteus.sh
proteus-mcp.cjs
```

Helper groups:

```text
memory/
  init target memory
  ingest sources
  record decisions
  query duplicates
  export markdown

labs/
  create realistic PoC workspaces
  validate configuration legitimacy
  capture negative controls
```
